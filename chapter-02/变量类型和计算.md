## 知识点
- 变量类型
	- 值类型 vs 引用类型
		- 值类型
			<pre>
			var a = 100
			var b = a
			a = 200
			console.log(b) // 100
			</pre>
		- 引用类型
			<pre>
			var a = {age:20}
			var b = a
			b.age = 21
			console.log(a.age) // 21
			</pre>
